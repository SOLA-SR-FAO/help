Capture Road Centrelines and Road Names

It is assumed that satellite imagery for the SR area has been made available to SOLA as another map layer through the publishing of the satellite imagery through a Geoserver setup on one of the SR desktop computers

 

Steps

  1. Select the Spatial Unit Editor menu option from the main Systematic Registration menu
  2. Zoom in to the area where you need to capture road centrelines and/or road names
  3. Select Road centreline (or place names) as the unit type from the dropdown list in the toolbar panel near the top of the screen
  4. For creating a new road centreline click on the     toolbar icon to the right of unit type drop down list and trace the line   to draw the road   with left mouse clicks and a double click to complete the road line. The new unit will appear
  5. Click on the   show/edit attributes icon to change the label for the new road centreline.
  6. Click on the  toolbar icon. The unit will disappear but can be returned by use of the toolbar icon. The new unit will now appear with a red line.
  7. If you need to make changes to existing unit, use the   toolbar icon and select (with a very positive left mouse click) the unit to be modified. Once you activated the edit mode, each node will appear as a red rectangle.
  8. Select    toolbar icon and move the activated nodes to make the required modifications.
  9. Save
  10. For adding new place names click on the    toolbar icon to the right of unit type drop down list and double click to add place locations. The new places will appear as orange square.
  11. Click on the  show/edit attributes icon to change the label for the new place names.
  12. Save

 

 

By means of    reset icon you can always Reset all editing in the client not sent to the server yet.
