Rights Claim

 

Initially Record a New Rights Claim


To record a new systematic registration Rights Claim (or Counter Claim), use the SOLA Desktop software and access the Application Details screen from the Applications > New Applications menu or by selecting
The initial details that need to be recorded are:

  • On the General tab for the new application form, in the Contact Details panel enter the claimant (or the person who is the contact person for the claimant or claimants) details including their first name, last name and address and contact details
  • On the Services tab, add a new service by double clicking on “Lodge SR Claim”
  • On the Property tab, enter the Parcel Number (in the Parcel Number field), StateLga?Ward name (in the Upi Ward Code field) and the land use
  • On the Documents tab, all documents such as the completed and signed claim form should be recorded (but there is no need to link the scanned image of the documents unless it is convenient to do so)
  • On the Fees tab, click on the Calculate Fees toolbar icon and record invoice number, amount paid and click on the Fees Paid radio button
  • Once the necessary information has been entered, save the application. This will lodge the application, assign an application number and display the Lodgement Notice for your review. You can choose to print (or save the Lodgement Notice and record and link it from the Documents tab)


Complete Claim Details


To complete the entry of the claim details you will need to edit the application that initially recorded the claim. Select the application in the assigned or unassigned application lists on the Dashboard and choose Edit. If the application is not displayed in the application lists, use the Search Application screen to locate it and click the Edit button. This will open the Application Details screen allowing you to edit the application details and save changes.

  1. Open the application and check the Fee paid flag on the Fees tab
  2. Assign the application with the Systematic Registration service to yourself
  3. Open the assigned application and in the Documents tab identify the documents without a link to a scanned image. At the same time and additional supporting documents not captured at the time the claim was originally recorded should be entered and linked to the scanned images of these supporting documents
  4. Click on the Document Search toolbar icon and search for the documents without linked imaged identified at the previous step. Select the document record, click on the icon and add the link for the scanned document
  5. Start the Systematic Registration Claim service
  6. SOLA will open the Property Details screen with the property reference for the new property already set. Verify this matches the claim form. If the property reference is not correct, close the Property Details screen without saving and remove the property details on the Property tab of the Application Details screen, save the application and click on to start the service again
  7. Leave the Area field blank as it will be automatically updated from the surveyed parcel details previously loaded
  8. On the Parcels tab, click Add parcel to open the Add Parcel dialog
  9. Use the Search parcel text field to check if a parcel matching the title reference exists in SOLA. If it does, click the result in the results list then click the Select button. Leave this form blank and return to it in another session, when the survey data has been loaded
  10. On the Rights / Restrictions tab you must add create the primary right for the property (typically ownership). Select the appropriate primary right type from the Right type drop down and click Create
  11. Enter the details for the primary right (e.g. claimant (to primary right) details in Ownership, Share and New Party Forms). Ensure the “Is primary right” checkbox is checked and Notation text (eg “Title created from systematic registration”) and leave the registration date blank. You will also need to Add a document to the primary right and add the Claim document
  12. If there are any other current rights or restrictions (i.e. easements, transmission (through inheritance court documents etc) noted during adjudication add an additional service for each right or restriction and add the details
  13. Save changes to Property Details and Complete the Systematic Registration service
